# USERAPI - Test Task

## REQUIREMENTS
- git
- Golang

## INSTALLATION

1. Simply clone this repository:

```
git clone https://github.com/faked86/userapi.git
```

2. Change directory to project directory:
```
cd userapi
```

3. Build executable file:
```
go build -o ./userapi ./cmd
```

## USAGE

1. Run executable:
```
./userapi
```
